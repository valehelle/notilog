A smarter changelog

Generate changelog from git merge and notify people when release

How it works?

Create a merge or pull request.

In the description, insert your changelog with the given format.

[CHANGELOG]
- Fix login bug
- Add twitter login feature

[NOTIFY]
adam@gmail.com
ITSupport@gmail.com
leader@gmail.com

Notilog will create a changelog from all the merge request.

Unreleased
- Fix login bug. [@alen]
- Add twitter login feature. [@alen]
- New visual identity. [@timothy]
- Fix button bug. [@timothy]
- Update Product UI. [@farhad]

Once you update the version name. Notilog will notify all the person you listed that you have a new released version.
